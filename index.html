Projects

Pricing Grid

A responsive site that I built using flexbox. It is a simple pricing table showcasing the pricing tier for a hosting company. Depending on how wide the viewport, these items either line up by row or by column.

Mondrian Painting

This is a site I created modeled after a Mondrian painting. It is made using CSS Grid to create and place the blocks of color that make these paintings so famous.

Tin Dog is a site that I built almost entirely using the third party framework Bootstrap. It is a mockup of a site for the app Tin Dog, Tinder for dogs.
